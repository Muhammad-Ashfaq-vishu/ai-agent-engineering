## 🧠 What I Learned

### Predictive AI

Predictive AI uses historical data to predict future outcomes.

Examples:

* Spam detection
* House price prediction
* Sales forecasting

Main concepts:

* Classification
* Regression
* Machine Learning

### Generative AI

Generative AI creates new content such as:

* Text
* Images
* Code
* Audio

Key concepts:

* Large Language Models (LLMs)
* Tokens
* Tokenization
* Context Windows

### Agentic AI

Agentic AI goes beyond generating text. It can:

* Plan tasks
* Use tools
* Make decisions
* Perform multi-step actions
* Work toward a goal autonomously

Examples:

* Research agents
* Coding agents
* Customer support agents

## 🔍 LLM Provider Comparison

| Provider  | Strengths                                                |
| --------- | -------------------------------------------------------- |
| OpenAI    | Strong ecosystem, GPT models, great developer experience |
| Anthropic | Claude models, excellent reasoning, AI safety focus      |
| Google    | Gemini models, multimodal capabilities, Google ecosystem |

### Conclusion

All three providers are powerful. The best choice depends on the project requirements, budget, and use case.

---

## 🚀 Key Takeaways

1. Predictive AI predicts outcomes from data.
2. Generative AI creates new content.
3. Agentic AI can take actions and solve tasks.
4. LLMs work by predicting the next token.
5. AI agents combine reasoning, planning, memory, and tools.
